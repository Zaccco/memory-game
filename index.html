Memory Game

This is some basic instructions for how to play memory!

  1. 1. Click on two different cards
  2. 2. If you find the same image behind both, you have found a match
  3. 3. If you find two different images they will flip back over and you will have to try again.
  4. 4. Keep trying to match images until all matches are found and the celebration alert pops up!
  5. 5. After the game is won the page will refresh with a new random order and you can play again.
